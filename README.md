## 🎮 About the Project

Minecraft3D is a lightweight voxel-style sandbox game inspired by Minecraft, developed in Python.  
It features procedural terrain generation using Perlin Noise and a first-person controller for exploration and building.

This project is still in **early development (Beta)** and will receive more updates in the future.

---

## ⚙️ Features

- 🌍 Procedural terrain generation (Perlin Noise)
- 🧱 Block placing and destroying system
- 👤 First Person Controller movement
- 🎯 Crosshair interaction system
- ⚡ Optimized chunk/subset terrain building
